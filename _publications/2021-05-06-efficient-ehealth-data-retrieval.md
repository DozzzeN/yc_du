### Contributions & Highlights
* **Triple Dictionary Index Structure:** Designed a novel index architecture tailored for attribute-value formatted Electronic Health Records (EHRs). This enables researchers to perform efficient encrypted searches using both **unique identities** and **multiple attribute values** as keywords simultaneously.
* **Oblivious Multi-User Retrieval:** Developed a privacy-preserving key management protocol where an identity server assists doctors in generating encryption keys via an **oblivious mechanism**. This allows authorized researchers to aggregate and search EHRs encrypted by different doctors without leaking sensitive data to the identity server.
* **Dynamic & Forward Security:** Achieved a high-security standard including **forward security** and **$L$-adaptive security**. The framework supports dynamic database operations (updates/deletions) while maintaining high computational efficiency, making it highly practical for large-scale cloud-assisted eHealth systems.
